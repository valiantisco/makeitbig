# UI Polish Skill

## Objective
Turn any UI into a clean, high-trust, modern interface that feels intentional and premium.

## Core Principles
- Fix layout before colors
- Fix spacing before adding elements
- Reduce clutter aggressively
- One clear focus per section

## What to Look For

### 1. Layout Issues
- Content too centered and narrow
- Not using horizontal space
- Sections feel boxed or cramped

### 2. Spacing Problems
- Inconsistent padding/margins
- Elements too close together
- No breathing room

### 3. Hierarchy Problems
- No clear visual order
- Everything feels same importance
- Weak headings

### 4. Trust Killers
- Generic styles
- Over-designed sections
- Too many colors
- Weak buttons

## Improvements to Apply

- Widen layouts
- Increase vertical spacing
- Strengthen headings
- Simplify sections
- Make primary CTA obvious

## Output Format

If UI code is provided:
- Return updated code
- Use find-and-replace style if possible

If visual feedback:
- Be direct
- Prioritize biggest impact changes first
