# Debug & Fix

## Objective
Quickly identify root issues and fix them cleanly.

## Approach

### Step 1: Identify Problem Type
- Rendering issue
- CSS/layout issue
- State/logic bug
- File/path issue

### Step 2: Find Root Cause
- Do not guess
- Look at structure and flow
- Check for common mistakes

### Common Issues

- Case-sensitive file paths (/Logo.svg vs /logo.svg)
- Incorrect imports
- Z-index / positioning bugs
- Overflow issues
- Broken flex/grid behavior

### Step 3: Fix Cleanly
- Minimal changes
- No hacks
- No unnecessary rewrites

## Output Format

- Explain issue briefly
- Provide exact fix
- Use replace-and-paste when possible
